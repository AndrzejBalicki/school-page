# Motyw „Przedszkole św. Józefa”

Autorski motyw blokowy dla WordPressa 6.6 lub nowszego. Łączy redakcyjną
typografię, pogodną paletę i lekką ilustrację SVG z prostym zarządzaniem
treścią w edytorze witryny.

## Instalacja ręczna

1. Skopiuj katalog `theme` do `wp-content/themes/school-page`.
2. W panelu WordPress przejdź do **Wygląd → Motywy**.
3. Aktywuj motyw **Przedszkole św. Józefa**.
4. Ustaw nazwę witryny i utwórz strony odpowiadające adresom we wzorcach.
5. Ustaw stronę statyczną jako stronę główną.

W WordPress Studio początkową strukturę i treści demonstracyjne tworzy
powtarzalny skrypt:

```bash
studio wp eval-file wp-content/seed-site.php
```

Skrypt nie nadpisuje istniejących stron ani wpisów, dlatego można uruchamiać
go ponownie bez utraty pracy redaktorów.

## Obecny zakres

- globalna paleta, typografia i odstępy w `theme.json`;
- responsywny nagłówek, pełnoekranowe menu mobilne i rozbudowana stopka;
- premium strona główna z autorską ilustracją;
- szablony strony, wpisu, archiwum i błędu 404;
- wzorce komunikatu, skrótów, wartości, aktualności i kontaktu;
- obsługa klawiatury, ograniczonych animacji i małych ekranów.

## Ważne

Treści i liczby użyte w wersji demonstracyjnej wymagają potwierdzenia przez
przedszkole. Przed publikacją trzeba również zweryfikować adresy, dane
kontaktowe, dokumenty prawne i zgody na ewentualne zdjęcia dzieci.
