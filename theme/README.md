# Motyw „Przedszkole św. Józefa”

Minimalny motyw blokowy dla WordPressa 6.6 lub nowszego.

## Instalacja ręczna

1. Skopiuj katalog `theme` do `wp-content/themes/school-page`.
2. W panelu WordPress przejdź do **Wygląd → Motywy**.
3. Aktywuj motyw **Przedszkole św. Józefa**.
4. Ustaw nazwę i logo witryny.
5. Utwórz menu oraz strony odpowiadające adresom użytym we wzorcach.
6. Ustaw stronę statyczną jako stronę główną.

## Obecny zakres

- globalna paleta, typografia i odstępy w `theme.json`;
- nagłówek z paskiem kontaktowym i nawigacją mobilną;
- stopka z kontaktem i ważnymi linkami;
- szablony strony głównej, strony, wpisu, archiwum i błędu 404;
- wzorce strony głównej;
- podstawowe style dostępności i responsywności.

## Ważne

Fonty `Nunito Sans` i `Source Sans 3` nie są jeszcze dołączone do repozytorium.
Do czasu ich legalnego, lokalnego osadzenia używane są fonty systemowe.

Treści we wzorcach są robocze. Przed wdrożeniem wymagają potwierdzenia przez
przedszkole, a odnośniki muszą odpowiadać utworzonym stronom WordPressa.
