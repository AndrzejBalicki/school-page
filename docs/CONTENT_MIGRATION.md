# Migracja treści

## Zasada

Migracja nie jest kopiowaniem całej starej strony. Każdy materiał otrzymuje
decyzję: zachować, poprawić, połączyć, zarchiwizować albo usunąć.

## Etap 1 — zabezpieczenie źródła

- pełna kopia bazy danych;
- kopia `wp-content/uploads`;
- kopia motywu i konfiguracji;
- eksport WXR z panelu WordPress;
- lista wszystkich publicznych adresów;
- zapis mapy przekierowań.

## Etap 2 — inwentaryzacja

Dla każdej strony lub wpisu zapisujemy:

| Pole | Znaczenie |
| --- | --- |
| Stary adres | Aktualny URL |
| Typ | Strona, aktualność, dokument, galeria |
| Tytuł | Obecna nazwa |
| Właściciel | Osoba odpowiedzialna za treść |
| Decyzja | Zachowaj, popraw, połącz, usuń |
| Nowy adres | Docelowy URL |
| Data przeglądu | Kiedy treść została potwierdzona |
| Przekierowanie | Wymagane 301 |
| Prywatność | Czy zawiera dane lub wizerunek dziecka |
| Dostępność | Czy dokument i media są dostępne |

## Etap 3 — treści priorytetowe

1. Dane kontaktowe i godziny pracy.
2. Rekrutacja.
3. Opłaty.
4. Kalendarium i ogłoszenia.
5. Jadłospis.
6. Standardy ochrony dzieci.
7. Informacje o placówce i ofercie.
8. Aktualności i wybrane galerie.

## Zdjęcia dzieci

Przed przeniesieniem zdjęcia trzeba potwierdzić:

- cel i miejsce publikacji;
- podstawę oraz aktualność zgody;
- możliwość wycofania zgody;
- brak informacji ujawniających więcej niż to potrzebne;
- brak krępującej, intymnej albo ryzykownej sytuacji;
- opinię dziecka odpowiednią do jego wieku;
- zasadność zachowania zdjęcia archiwalnego.

Brak pewności oznacza brak publikacji w publicznej galerii.

## Dokumenty

- preferujemy treść HTML;
- PDF ma mieć warstwę tekstową, nagłówki, kolejność czytania i opisowy tytuł;
- skany wymagają przepisania albo przygotowania dostępnego odpowiednika;
- nazwa linku opisuje dokument i format;
- dokument zawiera datę obowiązywania;
- niepublikowane wersje robocze pozostają poza biblioteką mediów.

## Przekierowania

Każdy wartościowy stary adres otrzymuje przekierowanie 301 do najbardziej
zbliżonej nowej treści. Nie przekierowujemy automatycznie wszystkich adresów
na stronę główną.
